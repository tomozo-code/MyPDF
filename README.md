# MyPDF（ともさんのPDF編集帖）

## 概要
C#で作成したPDF編集ツールです。
PDFの閲覧・編集・管理をシンプルに行うことを目的としています。

## 開発環境
* 言語：C#
* IDE：Visual Studio
* フレームワーク：.NET

## 主な機能
* PDFの閲覧
* しおり編集
* 保存

1. アプリを起動
2. PDFを開く
3. 編集して保存

## ビルド方法
1. リポジトリをクローン
