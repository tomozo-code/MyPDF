# ともさんのPDF編集帖（MyPDF）

## 概要
C#で作成したPDF編集ツールです。
PDFの閲覧・編集・管理をシンプルに行うことを目的としています。

## 開発環境
* 言語：C#
* IDE：Visual Studio
* フレームワーク：.NET

## 主な機能
* PDFの閲覧
* しおり編集
* 保存
