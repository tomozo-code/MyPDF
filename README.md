# ともさんのPDF編集帖（MyPDF）

## 概要
「ともさんのPDF編集帖」はC#で作成したPDF編集ツールです。
PDFの閲覧・編集・管理をシンプルに行うことを目的としています。

## 動作環境
* 日本語 Windows11（64bit）

## 開発環境
* 言語：C#
* IDE：Visual Studio
* フレームワーク：.NET

## 主な機能
* PDFの閲覧
* しおりの表示 / 編集
* プロパティの設定
* セキュリティの設定
* サムネイルの常時
* 保存

## 使い方
1. アプリ（MyPDF.exe）を起動
2. メニューからPDFファイルを開く
3. しおり・プロパティなどを編集
4. 保存して反映

## ビルド方法
1. リポジトリをクローン
2. Visual Studioで 'MyPDF.slnx' 開く
3. ビルド（Ctrl + shift + B）
4. 実行
