 # <img width="100" height="100" alt="icon" src="https://github.com/user-attachments/assets/3b9ef9d9-3f9c-4304-a03f-b941cf0952bc" />ともさんのPDF編集帖（MyPDF）
「ともさんのPDF編集帖」はC#で作成したPDF編集ツールです。

PDFの閲覧・編集・管理をシンプルに行うことを目的としています。

---

## ✨ 概要
PDFの閲覧・しおり編集・プロパティ確認などを直感的に操作できるデスクトップアプリです。

---

## 🖥 動作環境
- 日本語 Windows11（64bit）

---

## 💻 開発環境
- 言語：C#
- IDE：Visual Studio
- フレームワーク：.NET

---

## ✨ 主な機能
- 📄 PDFの閲覧
- 📑 しおりの表示 / 編集 / 移動
- 📚 しおりの色 / スタイル
- 📋 ページの移動 / 挿入 / 置換 / 抽出 / 回転 / 削除
- 📝 プロパティの設定
- 🔐 セキュリティの設定
- 💾 保存

---

## 📊 対応機能一覧

| 機能 | 対応 | 内容 |
|------|------|------|
| 📖 PDF閲覧 | ✅ | 通常表示・ページ移動 |
| 📑 しおり表示 | ✅ | PDF内のしおりを表示 |
| 📑 しおり編集 | ✅ | 作成・削除・移動・名称変更 |
| 📚 しおり色・スタイル | ✅ | 色・太字・イタリック対応 |
| 📋 ページ移動 | ✅ | 任意範囲の移動 |
| ➕ ページ挿入 | ✅ | 他PDFから挿入 |
| 🔄 ページ置換 | ✅ | 指定範囲を別PDFで置換 |
| ✂ ページ抽出 | ✅ | 指定範囲を新規PDFへ出力 |
| 🔃 ページ回転 | ✅ | 90度単位で回転 |
| 🗑 ページ削除 | ✅ | ページ削除 |
| 📝 プロパティ編集 | ✅ | タイトル・著者など |
| 🔐 セキュリティ設定 | ✅ | パスワード設定・権限制御 |
| 💾 保存 | ✅ | 上書き保存 |
| 📂 既存のPDFアプリで開く | ✅ | Acrobat Readerなどの既定アプリで開く |
| 🗂 サムネイル | ❌ | サムネイル表示未対応 |
| 📝 テキスト編集 | ❌ | PDF内テキストの直接編集不可 |
| 🖼 画像編集 | ❌ | 画像の編集不可 |
| ✍ 注釈（コメント） | ❌ | コメント・ハイライト未対応 |
| 📕 PDF作成 | ❌ | PDFファイルの作成未対応 |
| 🖨 PDF印刷 | ❌ | PDFファイルの作成未対応 |
| 📄 OCR機能 | ❌ | OCR機能未対応 |

- ※注釈は表示されませんが、注釈データは保持されています。

## 🚀 使い方
1. アプリ（MyPDF.exe）を起動
2. メニューからPDFファイルを開く
3. しおり・プロパティなどを編集
4. 保存して反映

---

## 📦 ビルド方法
1. リポジトリをクローン
2. Visual Studioで 'MyPDF.slnx' 開く
3. ビルド（Ctrl + shift + B）
4. 実行
